Delain¶

“Delain” is a the code name for a Traffic Server plugin until I find a more descriptive name. This plugin will be a substitute for both the cookie remap and header rewrite plugins. The primary goals of the project are

  • A YAML based plugin for these functions.

  • Demonstrate Canned-YAML as a YAML configuration support library.

  • Demonstrate the utility of libswoc++.
